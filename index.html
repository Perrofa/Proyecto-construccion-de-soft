¿Cuál es la diferencia entre Internet y la World Wide Web?
El Internet es una red global de computadoras que nos permite comunicarnos y mandar informacion, mientras que la World Wide Web es un servicio que nos permite el acceso a documentos mediante hipervínculos
¿Cuáles son las partes de un URL?
Primero que nada viene el esquema que indica el protocolo que se esta usando, despues viene el nombre del dominio, la ruta que es donde se indica la estructura de los archivos y directorios del servidor, luego vienen los parametros que sirven para que se puedan añadir datos adicionales, finalmente estan los fragmentos que se usan para especificar una parte que se accede
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
get sirve para solicitar un recurso en especifico al servidor, head por el contrario solo recibe los encabezados de los recursos, post envía datos al servidor, put se usa para crear o remplazar datos en el servidor, patch se usa para aplicar modificaciones, y finalmente delete sirve para eliminar
